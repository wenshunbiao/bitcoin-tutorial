深入区块链
===============

## 导言

区块链（Blockchain）技术源于比特币，本质上其实是一个分布式的，不可篡改的数据库，具有可验证、可信任的特性。

本系列教程带你逐步深入理解区块链技术，并附带源码教程（主要使用PHP编写），但需要你对密码学原理、安全机制、共识算法、哈希算法有一定的认知。

## Install
```
git clone https://github.com/wenshunbiao/bitcoin-tutorial.git
```
```
composer install
```
